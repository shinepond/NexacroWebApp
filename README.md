# NexacroWebApp
Nexacro Web App 을 띄우기 위한 android native Application

## 1. gradle setting
 - Android Gradle Plugin Version : 7.2.0
 - gradle version : 7.4

## 2. Android Sdk Version
 - minimum Sdk 21 
 - target Sdk 32
 - compile Sdk 32

## 3. Build Variants
 - Build types
    1) release (디버깅 불가)
    2) debug (디버깅 허용)
 
 - Flavors - server
   1) dev 
   2) prod 
   3) local 

## 4. dependencies
 - 넥사크로 라이브러리 : project/app/libs/nexacro.aar
